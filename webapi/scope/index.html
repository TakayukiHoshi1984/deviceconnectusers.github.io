スコープについて

DeviceConnectで指定できるスコープは以下の通りです。詳細は徐々に記入していきます。

スコープ名 説明 APIドキュメント
servicediscovery 利用できるデバイスを検索します サービス検索API
serviceinformation サービスに関する情報を取得します サービス情報API
system DeviceConnectシステム情報を提供します DeviceConnectシステムの情報API
battery バッテリー情報を取得します バッテリーAPI
connection Bluetooth/BLE/WiFi/NFCの接続状態の取得、設定を行います コネクトAPI
deviceorientation スマートデバイスの加速度センサー機能を提供します デバイス向き/加速度API
filedescriptor 新しいファイルを作成したり、内容の書き込み、読み込みを行います。 ファイルストリームAPI
file デバイス内のファイル一覧を取得したり、ファイル操作を行います。 ファイルAPI
mediaplayer スマートデバイス上のメディアの再生状態の変更要求を通知します メディアプレイヤーAPI
mediastreamrecording デバイスのカメラを使って録画、プレビューを行います。 メディアストリーミング録画API
notification デバイスに通知を送ります。 通知API
phone 周辺機器からスマートフォンの通話機能もしくは通話操作機能を提供します 電話API
proximity スマートデバイスの近接センサーの物/人の検知を通知します 近接センサーAPI
setting スマートデバイスの各種設定状態の取得および設定機能を提供します 設定API
vibration バイブレーションを実行します バイブレーションAPI
light LEDライトの明るさ、色を設定します ライトAPI
remotecontroller スマートデバイスが保持する赤外線などの信号の送受信機能を提供します リモートコントローラーAPI
drivecontroller スマートデバイスの駆動処理を提供します デバイス駆動API
mhealth mhealth対応の健康機器の数値を取得します  
sphero Spheroに搭載されているセンサーの値を通知します SpheroセンサーAPI
dice DICE+に搭載されているセンサーの値を通知します  
temperature 温度を測定します 温度API
camera スマートデバイス(Cameraデバイスなど）のズーム操作機能を提供します カメラ操作API
canvas デバイスに画像を描画・削除する機能を提供します Canvas API
health ヘルスケア関連のデバイスから通知されるイベントを受信します ヘルスケア API
touch スマートデバイス上でのタッチイベントを通知します タッチAPI
humandetection スマートデバイスの人体検出機能を提供します 人体検出API
keyevent デバイスのキーイベントが操作されたことを通知します キーイベントAPI
omnidirectionalImage 全方位パノラマ画像をROI画像へ変換する機能を提供します 全方位パノラマ画像操作API
tv スマートデバイスが持つTV機器を操作する機能を提供します TV操作API
powermeter 電力量を取得したり、電気利用のオンオフを設定します 電力量 API
humidity 湿度を測定します 湿度API
illuminance 照度を測定します 照度API
videochat スマートデバイスが持つビデオチャットを操作する機能を提供します VideoChat API
airconditioner スマートデバイスが持つエアーコンディショナー機器を操作する機能を提供します エアコンAPI
gpio General Purpose Input/Output（汎用入出力）にアクセスする機能を提供します GPIO API
geolocation スマートデバイスの位置情報機能を提供します 位置情報API
device スマートデバイスへの接続関連の機能を提供します デバイス接続切断API
ecg デバイスのECGのイベントを通知します ECG API
echonetlite スマートデバイスがサポートしているECHONET Lite 機器オブジェクトプロパティの設定内容を操作します ECHONETLite API
messagehook メッセージ送受信の機能を提供します MessageHook API
power スマートデバイスの電源状態を操作します 電源状態操作API
poseestimation デバイスの姿勢推定のイベントを通知します 姿勢推定API
stressestimation ストレス推定のイベントを通知します ストレス推定API
walkstate 歩行状態を通知する機能を提供します 歩行推定API
