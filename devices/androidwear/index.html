AndroidWearプラグインの使い方

AndroidWearは、Googleが発表した腕時計型ウェアラブルデバイス向けのプラットフォームです。 AndroidWearプラグインは、AndroidWearをサポートしている腕時計型デバイス上で動作するプラグインになります。

セットアップ

まずスマートフォンにAndroidWearアプリ（[AndroidWear](https://play.google.com/store/apps/details?id=com.google.android.wearable.app&hl=ja）をインストールします。

インストールしたらアプリの説明に従ってAndroidWearのセットアップを行います。

AndroidWearアプリ上で以下の画面になれば完了です。

プラグインのインストール

AndroidWear用のDeviceConnectプラグインをビルドするか、Androidであればビルドしたapkファイルを取得、インストールしてください。

プラグインとデバイスの接続

AndroidWear側のデータをAndroidWearプラグインで取得する場合には、AndroidWear側でAndroidWearプラグインのアプリをインストールする必要があります。 まず、AndroidWear側でGooglePlayを開いてください。

初回起動時は、GooglePlayの利用規約が表示されます。

下にスクロールして、共有するをタップしてください。

GooglePlayのトップ画面が表示されるので、再びスクロールし、AndroidWear (Device Connect Device Plug-in)をインストールしてください。

インストールが完了すると、DeviceConnectのリクエストでAndroidWearが操作できるようになります。

リクエストが成功しない場合

上記手順のみでは、Canvas,Touch,KeyEventが動作しない場合があります。 その場合は、AndroidWear側で以下のアプリがインストールされているかを確認してください。

確認方法は、AndroidWear側でGooglePlayを開いたのちにNotificationを表示します。 その中から、左側のアイコンをタップしてください。

インストール中・インストール済みアプリの一覧画面が表示されます。 その中から、「時計」と「Android Wear - 時計」というアプリのアップデートが終わってからDeviceConnectのリクエストを送信してみてください。

プラグインとデバイスの切断

AndroidWearは、BluetoothとWiFiでAndroid端末とデータをやりとりするため、WiFiとBluetoothをOFFにしないと切断ができない。 その場合でも、WiFiとBluetoothをONに戻すことで自動的に接続されます。 完全に切断する場合は、AndroidWearの設定画面から、リセットを行ってください。

プラグインがサポートしている機能

プラグインがサポートしている機能は次の通りです。

  • Canvas API
  • デバイス向き/加速度API
  • キーイベントAPI
  • 通知API
  • タッチAPI
  • バイブレーションAPI
