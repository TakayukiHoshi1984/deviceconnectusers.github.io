F-PLUGプラグインの使い方

F-PLUGプラグインは、コンセントに挿すだけで、家庭の電気機器の電気使用量、周辺の温度、湿度、および照度を計測できます。

セットアップ

F-PLUGをコンセントにさします。 その後、AndroidのBluetooth設定画面でF-PLUGとの接続を行います。

ペアリングは、こちらのドキュメントを参考に行ってください。

プラグインのインストール

F-PLUG用のDeviceConnectプラグインをビルドするか、Androidであればビルドしたapkファイルを取得、インストールしてください。

プラグインとデバイスの接続

まず、F-PLUGの設定画面を開き、追加ボタンを押したのちに３ページ目までフリックしてください。 ここまでで、F-PLUGとペアリングがすんでいない場合は、ペアリングを行ってください。 ペアリングが完了している場合は、「F-PLUGの検索と接続」ボタンをタップしてください。

接続されているF-PLUGが表示されるので、その文字が赤から青に変化するのを待ってからF-PLUGをタップしてください。

F-PLUGのデータを取得するための初期化を行うために、F-PLUG初期化ボタンをタップしてください。

確認のダイアログが表示されるので、OKをタップしてください。

初期化に成功すると、成功と表示されます。

ここまで完了すると、DeviceConnectでリクエストとレスポンスのやりとりができるようになります。

プラグインとデバイスの切断

プラグインとデバイスの切断を行う場合は、AndroidのBluetooth設定画面でF-PLUGとの切断を行います。

プラグインがサポートしている機能

プラグインがサポートしている機能は次の通りです。

  • 湿度API
  • 照度API
  • Light API
  • 電力メーターAPI
  • 温度 API
